MAX

This limits usage for a group or user. The MAX option cannot be used to limit license borrowing.

Usage:

MAX num_lic feature [: keyword = value ] type { name | group_name }

Term

Description

num_lic

Usage limit for this user or group.

feature

Feature this limit applies to.

type

One of the following: USER, HOST, DISPLAY, INTERNET, PROJECT, GROUP, or HOST_GROUP.

name

Name of an item of type type for which usage is limited.

group_name

Name of the group to limit. Group names are case sensitive.

Example:

MAX 5 f1 USER jan

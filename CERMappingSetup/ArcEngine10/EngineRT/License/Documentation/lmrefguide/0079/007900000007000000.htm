Starting and stopping the License Manager

  1. Launch the ArcGIS License Server Administrator from Start > Programs > ArcGIS > License Manager > License Server Administrator.
  2. Select Start/Stop License Service in the table of contents using the Start, Stop, or Restart buttons to manage the license service.

From the command line, switch to the License Manager installation location (the default installation location is c:\program files\arcgis\license10.0\bin) and run the following commands:

  • To start the License Manager:
    c:\program files\arcgis\license10.0\bin> lmgrd -z -c service.txt

  • To stop the License Manager:
    c:\program files\arcgis\license10.0\bin> lmutil lmdown -c service.txt

ArcGIS License Manager

The version 11.6.1 FLEXnet Publisher license management software must be installed to manage ArcGIS Desktop Concurrent Use, ArcGIS Engine Runtime Concurrent Use, and ArcInfo Workstation. In an enterprise environment, ArcGIS License Manager can also host and distribute enterprise single use licenses.

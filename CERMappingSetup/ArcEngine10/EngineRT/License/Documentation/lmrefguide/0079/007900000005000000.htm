License Manager Installation and Startup

ArcGIS 10 uses FLEXnet Publisher License Manager v11.6.1. This version of the license manager will support ArcGIS 10 and ArcGIS 9.x releases. Visit the Existing users page for information on setting up different versions of the license manager.

The ArcGIS License Manager communicates through TCP/IP, which must be installed and functioning properly on your Windows license server. In addition, TCP/IP requires that either a network card, along with its drivers, or the MS Loopback Adapter be installed on your Windows machine.

An Internet connection is also strongly recommended for the license authorization process. If you do not have an internet connection on the license manager machine, you can select the e-mail or website authorization option on the Authorization Method panel.

Upon purchasing your ArcGIS products, you will receive an e-mail from ESRI Customer Service listing the purchased products and associated authorization numbers and license counts. Follow the steps below to install, authorize, and start the license manager:

1. Run the License Manager setup from your ArcGIS Installation media.

2. Follow the instructions to install the license manager to the desired location.

3. At the end of the installation, the ArcGIS License Server Administrator appears, allowing you to complete the authorization process and start the license service. If you choose to do this step at a later time, on Windows, you can access the License Server Administrator from Start > Programs > ArcGIS > License Manager > License Server Administrator. On UNIX/Linux, you can run the License Server Administrator from the installation location using the following command:

[ install_path ]/arcgis/license10.0./LSAdmin

4. Select the Authorization folder in the table of contents and click the Authorize Now button to launch the Software Authorization Wizard.

5. Follow the instructions on the dialog boxes to complete the authorization process.

The initial Web site or e-mail authorization option for this version of the License Manager is a two-step process. Work is being done with the software vendor to make this a one-step process similar to the Single Use software authorization in an upcoming release. Once the initial configuration is completed and licenses authorized, subsequent authorizations are only a single step process.

6. Click Finish to close the Wizard and return to the License Server Administrator.

7. Click the Start/Stop License Service folder in the table of contents and click the Start button to start the license service.

Contacting ESRI Customer Service

If for any reason you are not able to locate your authorization information, you can contact ESRI Customer Service and request that they resend it.

In the United States, you can contact ESRI Customer Service by calling 888-377-4575 or visit the ESRI Customer Service site.

Outside the United States, contact your local ESRI distributor. For information regarding your local distributor, visit the Web site at http://www.esri.com/company/contact.html.

Using the ArcGIS License Server Administrator

The License Server Administrator can be accessed from Start > Programs > ArcGIS > License Manager > License Server Administrator.

On UNIX/Linux, the License Server Administrator can be launched from the license manager installation location:

[ installation_path ]/arcgis/license9.4 ./LSAdmin

The following operations can be performed using the License Server Administrator:

  • Starting and stopping the license manager
  • Checking the license server status
  • Configuring license borrowing
  • Viewing the audit log
  • Viewing license availability
  • Authorizing additional features
  • Diagnosing errors

EXCLUDEALL

This excludes a user or predefined group of users from the list of who is allowed to use all features on the license manager.

Usage:

EXCLUDEALL type {name | group_name}

Term

Description

type

One of the following: USER, HOST, DISPLAY, INTERNET, GROUP, HOST_GROUP.

name

Name of an item of a type above for which license usage is excluded.

group_name

Name of the group to exclude. Group names are case sensitive.

Example:

To exclude any user on the system called test from all features on the license manager.

EXCLUDEALL HOST test

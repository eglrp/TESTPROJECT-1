Client machine setup—The ArcGIS Administrator

The ArcGIS Administrator utility is only installed with the ArcGIS Desktop, ArcGIS Engine Runtime, and ArcGIS Engine Developer Kit software and is launched after the installation of the software.

If you would like to access it at a later time, you can launch it from Start > Programs > ArcGIS > ArcGIS Administrator. On UNIX/Linux, the ArcGIS Administrator can be launched from the product installation location using the following command:

[install path]/arcgis/engine9.4 ./authorizeSoftware

The ArcGIS Administrator allows you to define both the concurrent use and single use environments. This help document only provides information regarding the concurrent use case. For setting up the single use environment, consult the ArcGIS Administrator Help, which can be accessed from the ArcGIS Administrator Help button.

The ArcGIS Administrator allows you to

  1. Select the language to use for your ArcGIS Desktop and ArcGIS Engine environments.
  2. Select a product to use and define a license manager.
  3. View license availability.
  4. Borrow available concurrent use licenses.
  5. Return borrowed licenses.
  6. Add and remove Data licenses.

Select a language

  1. Launch the ArcGIS Administrator.
  2. Click the top node in the table of contents.
  3. Select the desired language.
  4. Click Apply to set your changes without closing the ArcGIS Administrator or click OK to apply changes and close the ArcGIS Administrator.

Select a product and define a license manager

  1. Launch the ArcGIS Administrator and select the desired product folder (Desktop or Engine).
  2. Select the button next to the concurrent use product you would like to use.
  3. Click Change, enter the name of your license manager, then click OK. The name of the License Manager is the host name of the machine where the License Manager is installed.
  4. Click OK on the Change dialog box.
  5. Click Apply to set your changes without closing the ArcGIS Administrator or click OK to apply changes and close the ArcGIS Administrator.

View license availability

  1. Launch the ArcGIS Administrator and select the desired product folder (Desktop or Engine).
  2. Select the desired concurrent use product.
  3. Select Availability to view licenses available on the License Manager selected in the product folder. The license expiration date is also provided.

Borrow licenses

Note: The default borrow timeout value is 30 days. To modify this value, see the Configuring license borrowing topic in the Administering the License Manager section.

  1. Launch the ArcGIS Administrator and select the desired product folder (Desktop or Engine) for which you want to borrow licenses.
  2. For ArcGIS Desktop, select the concurrent use license level you would like to borrow—ArcInfo, ArcEditor, or ArcView—or select the ArcGIS Engine Runtime (Concurrent Use) from the Engine folder. Ensure that the license manager has been defined.
  3. Select the Borrow/Return folder.
  4. Check the boxes of the features you want to borrow and click Apply.
  5. After the licenses have been borrowed, click OK to close the ArcGIS Administrator. You can now use your ArcGIS software while off the network.

Return borrowed license

Note: To return licenses manually, you first need to connect to your network.

  1. Launch the ArcGIS Administrator and ensure that you have the same product and license manager set in the product folder.
  2. Select the Borrow/Return folder.
  3. Uncheck the boxes next to the features you want to return and click Apply.
  4. Click OK to close the ArcGIS Administrator.

Existing users

9.x and 10 License Managers

If you are currently using a License Manager released prior to ArcGIS 10, you first need to uninstall this older version to proceed with the installation of the ArcGIS License Manager 10. License files at version 9 or earlier will not work with this new version of the License Manager. You will need to first authorize licenses on the license server using the new authorization information sent via e-mail by ESRI Customer Service before starting the license service.

Contacting ESRI Customer Service

If for any reason you are not able to locate your authorization information, you can contact ESRI Customer Service and request that it be re-sent.

In the United States, you can contact ESRI Customer Service by calling 888-377-4575 or visit the ESRI Customer Service site.

Outside the United States, contact your local ESRI distributor. For information regarding your local distributor, visit the Web site at http://www.esri.com/company/contact.html.

Authorizing additional features

Note:

To perform this operation, you need to refer to the authorization information sent via e-mail by ESRI Customer Service.

  1. Launch the ArcGIS License Server Administrator from Start > Programs > ArcGIS > License Manager > License Server Administrator.
  2. Click Authorization in the table of contents.
  3. Click Authorize Now to launch the Software Authorization Wizard.
  4. Follow the instructions to complete the authorization process and return to the ArcGIS License Server Administrator.
  5. In the License Server Administrator, open the Start/Stop License Service folder in the table of contents.
  6. Click Re-read Licenses to refresh the licenses available for use.

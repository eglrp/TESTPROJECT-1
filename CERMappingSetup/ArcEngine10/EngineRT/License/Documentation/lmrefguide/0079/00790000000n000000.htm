Viewing license availability

  1. Launch the ArcGIS License Server Administrator from Start > Programs > ArcGIS > License Manager > License Server Administrator.
  2. Choose Availability in the table of contents.

    The features that you have licensed are listed in the table on the right along with version, total, and available counts.

  3. Double-click a feature to get information on the authorization number, expiration date, number of licenses, and number of licenses borrowed.

Based on your license agreement with ESRI, you might have the option to select and view the availability of single use licenses on your license manager.

Viewing the audit log

The audit log captures license borrow, return, and enterprise single use authorization activity. You can view the log file using the ArcGIS License Server Administrator.

  1. Launch the ArcGIS License Server Administrator from Start > Programs > ArcGIS > License Manager > License Server Administrator.
  2. Choose Configure in the table of contents.
  3. Under the Audit Log path, click the View button to open a dialog box that displays the license borrow activity.

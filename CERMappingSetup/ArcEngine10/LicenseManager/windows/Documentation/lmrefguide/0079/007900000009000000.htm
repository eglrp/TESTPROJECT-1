Configuring license borrowing

License borrowing allows users of ArcGIS Desktop and ArcGIS Engine Runtime to temporarily take concurrent use features from the license manager and install them locally on their machines to use when off the network.

  1. Launch the ArcGIS License Server Administrator from Start > Programs > ArcGIS > License Manager > License Server Administrator.
  2. Select Configure from the table of contents.
  3. Click the Allow License Borrowing check box if unchecked.
  4. Enter the maximum number of days for which licenses can be borrowed. Shorter borrow timeouts are recommended.
  5. Click Apply to apply changes and restart the service.

DEBUGLOG

This specifies a location for the debug log output from the vendor daemon associated with this options file. Preceding the debug_log_path with a + character appends logging entries; otherwise, the file is overwritten each time the daemon is started. Note that this affects output from only the vendor daemon associated with this options file.

DEBUGLOG [+] debug_log_path

Example:

DEBUGLOG + C:\temp\lmgrd.log

TIMEOUT and TIMEOUTALL

Sets the time after which a hung license is released and reclaimed by the vendor daemon

Usage:

TIMEOUT feature seconds

Examples:

To return a hung ArcInfo license after 10 seconds, use TIMEOUT:

TIMEOUT ARC/INFO 10

To return any hung license after a certain time period, use TIMEOUTALL:

TIMEOUTALL seconds

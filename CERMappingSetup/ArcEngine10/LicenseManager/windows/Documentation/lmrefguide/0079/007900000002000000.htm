Welcome

This guide is designed to help you perform tasks common to license management. The primary audience for this guide is any site administrator responsible for installing and managing ArcGIS Desktop, ArcGIS Engine, and ArcInfo Workstation. This documentation will assist you in using the ArcGIS License Manager 10 and troubleshoot issues that may arise during its use. In the table of contents on the left, click the topic you want to learn more about. You can also search for a particular topic using the Search window.

How to use this document

This document is divided into the sections below:

Getting started

The Getting started section provides installation and startup information that will help you get the ArcGIS License Manager 10 running. It also describes the process to set up the client machines to use the license manager.

Administering the License Manager

The Administering the License Manager section provides information that will assist you in using the ArcGIS License Manager 10 and managing your licenses. This section provides information on configuring license borrowing and viewing license availability.

Advanced topics

Refer to this section for complex license management issues, such as using the options file or failover license servers.

Troubleshooting

This section lists common problems with the License Manager, probable causes, and suggested solutions. For example, if you have trouble connecting to a Windows license server, you could go to the topic Cannot connect to license server.

Glossary

The glossary contains definitions of commonly used ArcGIS copy protection terms.

Contacting ESRI Customer Service

If for any reason you are not able to locate your authorization information, you can contact ESRI Customer Service and request it to be sent again.

In the United States, you can contact ESRI Customer Service by calling 888-377-4575 or visiting the ESRI Customer Service site https://service.esri.com.

Outside the United States, contact your local ESRI distributor. For information regarding your local distributor, visit the Web site at http://www.esri.com/company/contact.html.

Technical Support Knowledge Base

If you are unable to find the information you need in this guide, refer to the ESRI Technical Support Knowledge Base, http://support.esri.com.

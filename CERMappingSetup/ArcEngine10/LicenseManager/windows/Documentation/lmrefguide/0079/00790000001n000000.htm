RESERVE

Use the RESERVE line to reserve licenses for a specific user or group

Usage:

RESERVE num_lic feature [: keyword = value ] type { name | group_name }

Term

Description

num_lic

Number of licenses to reserve for this user or group.

feature

Feature this reservation applies to.

type

One of USER, HOST, DISPLAY, INTERNET, GROUP, or HOST_GROUP.

name

Name of an item of type type for which license usage is reserved.

group_name

Name of the group for which license usage is reserved. Group names are case sensitive.

Example:

RESERVE 1 f1 USER mel

If you want to reserve a license for each of several users or groups, you must use a separate RESERVE line for each user or group.

GROUPCASEINSENSITIVE

Usage:

GROUPCASEINSENSITIVE OFF/ON

By default, GROUPCASEINSENSITIVE is OFF, and user names and host names are treated as case sensitive. If it is set to ON, user names and host names specified in the options file under GROUP and HOST_GROUP are defined as case insensitive.
